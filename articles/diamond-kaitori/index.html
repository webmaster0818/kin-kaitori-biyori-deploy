【2026年最新】ダイヤモンド・宝石買取ガイド — 4Cの評価と高く売るコツ

ダイヤモンドの4C — 買取価格を決める4つの評価基準

ダイヤモンドの品質と価値を決めるのが4Cです。これはGIA（米国宝石学会）が策定した世界共通の評価基準で、以下の4つの要素から構成されます。

1. Carat（カラット）— 重量

カラットはダイヤモンドの重量を表す単位です。1カラット = 0.2グラムです。カラット数が大きいほど希少性が高く、価格も高くなります。

カラット直径の目安買取価格の目安（中品質の場合）
0.1ct約3.0mm約3,000〜10,000円
0.2ct約3.8mm約10,000〜30,000円
0.3ct約4.3mm約30,000〜80,000円
0.5ct約5.2mm約80,000〜200,000円
0.7ct約5.8mm約150,000〜400,000円
1.0ct約6.5mm約300,000〜1,000,000円
2.0ct約8.2mm約1,000,000〜5,000,000円
3.0ct以上約9.3mm以上数百万〜数千万円

カラット数が2倍になると、価格は2倍以上になります。これは大きなダイヤモンドほど希少性が高いためです。特に1カラットの壁は大きく、0.99ctと1.00ctでは心理的な価格差が生じます。

2. Cut（カット）— 輝き

カットはダイヤモンドの輝きを決める最も重要な要素です。どれだけ大きくても、カットが悪ければ輝きが鈍くなります。

グレード評価買取価格への影響
Excellent（エクセレント）最高品質最も高い
Very Good（ベリーグッド）優秀高い
Good（グッド）良好標準的
Fair（フェア）やや劣るやや低い
Poor（プア）不良低い

3EX（トリプルエクセレント）は、カット・対称性・研磨の3項目すべてがExcellentのダイヤモンドで、最も高い評価を受けます。ハート＆キューピッドパターンが見えるものは特に人気があります。

3. Clarity（クラリティ）— 透明度

クラリティは、ダイヤモンド内部の内包物（インクルージョン）や表面の傷（ブレミッシュ）の少なさを表します。

グレード意味買取価格への影響
FL（フローレス）10倍拡大で内外無傷最高（非常に希少）
IF（インターナリーフローレス）10倍拡大で内部無傷非常に高い
VVS1-VVS210倍拡大でも発見困難な微小内包物高い
VS1-VS210倍拡大で軽い内包物標準〜やや高い
SI1-SI210倍拡大で容易に発見できる内包物標準
I1-I3肉眼で見える内包物低い

4. Color（カラー）— 色

ダイヤモンドのカラーは、D（完全な無色）からZ（薄い黄色）までのアルファベットで表されます。無色に近いほど高い評価を受けます。

グレード色の特徴買取価格への影響
D-E-F無色（Colorless）最も高い
G-H-I-Jほぼ無色（Near Colorless）高い〜標準
K-L-Mわずかに黄色（Faint）やや低い
N-R非常に薄い黄色（Very Light）低い
S-Z薄い黄色（Light）低い

ただし、Z以上の濃い色を持つ「ファンシーカラーダイヤモンド」（ファンシーイエロー、ファンシーピンク、ファンシーブルーなど）は、むしろ希少性が高く高額で取引されます。

一括査定で最高値を調べる（ヒカカク！）近くの店舗で査定する（買取大吉）出張買取を申し込む（ウリエル）好きな方法で査定する（ティファナ）

4Cランク別の買取価格差 — 同じ1ctでもこれだけ違う

4Cの組み合わせによって、同じ1カラットのダイヤモンドでも30万円〜200万円以上の価格差が出ます。以下に代表的な組み合わせによる価格差を示します。

ランク4Cの組み合わせ（1ct）買取価格の目安
最高ランクD / Excellent / IF / 1.0ct約150〜200万円
高ランクE / Very Good / VVS1 / 1.0ct約100〜150万円
中ランクG / Good / VS1 / 1.0ct約60〜100万円
標準ランクI / Good / SI1 / 1.0ct約30〜60万円
低ランクK / Fair / I1 / 1.0ct約10〜30万円

このように、同じ1カラットでも最高ランクと低ランクでは10倍以上の差が出ます。だからこそ鑑定書（グレーディングレポート）が重要なのです。

メレダイヤ（小粒ダイヤ）の買取について

0.1カラット以下の小さなダイヤモンドをメレダイヤと呼びます。パヴェセッティングやエタニティリングに多数使われている小粒のダイヤモンドです。

メレダイヤのサイズ1石あたりの買取価格備考
0.05ct以下数十円〜数百円ほぼ値段がつかない
0.05〜0.1ct数百円〜1,000円多数あれば合算で数千円に
0.1〜0.2ct1,000〜5,000円品質による差が大きい

メレダイヤ自体は高額にはなりませんが、台座の金やプラチナの素材価値が主な買取価格の構成要素になります。例えば、メレダイヤが20石付いたK18リング（5g）なら、金の素材価値（約57,000円）+ メレダイヤ（数千円）= 約6万円前後になります。

ポイント：「メレダイヤだから値段がつかない」とあきらめず、台座の金属価値を含めて査定してもらいましょう。

フリマ・ヤフオクが有利なケース：鑑定書付きの高品質ダイヤ（1ct以上、Gカラー以上）で、購入証明書もあるブランドジュエリーの場合。信頼性を担保できるなら、個人売買の方が高値がつくことがあります。

買取業者が有利なケース：鑑定書なし、メレダイヤ、ノーブランド品、壊れたジュエリーなど。トラブルリスクなく確実に現金化できます。

鑑定書の有無による価格差

ダイヤモンドの買取において、鑑定書（グレーディングレポート）の有無は大きな価格差を生みます。

鑑定書の状態買取価格への影響補足
GIA鑑定書あり最も高い（基準価格）世界最高権威の鑑定機関
CGL鑑定書あり高い（GIAに次ぐ）日本国内で最も信頼される鑑定機関
その他の鑑定書あり標準的AGTなど
鑑定書なし10〜30%低下業者が独自に鑑定するためリスク分が差引かれる

鑑定書がない場合でも買取は可能ですが、業者が独自に鑑定するため、「グレードが低く評価される」リスクがあります。鑑定書があれば客観的なグレードが証明されているため、高い評価を受けやすくなります。

鑑定書を紛失した場合：GIAやCGLに鑑定書の再発行を依頼できる場合があります。ダイヤモンドにレーザー刻印されているGIAのレポート番号がわかれば、GIAのデータベースで鑑定結果を確認できます。

鑑定書を再発行してから売るべきか — 損得計算

「鑑定書を紛失したけど、再発行してから売った方がいい？」という質問は多いです。結論はダイヤモンドのサイズと品質によるです。

ダイヤモンドのサイズ鑑定書再発行の費用鑑定書の有無による価格差再発行すべきか
0.3ct以下約5,000〜10,000円約3,000〜10,000円不要（元が取れない場合が多い）
0.3〜0.5ct約5,000〜10,000円約10,000〜50,000円品質次第（高品質なら再発行推奨）
0.5〜1.0ct約8,000〜15,000円約50,000〜200,000円再発行推奨
1.0ct以上約10,000〜20,000円約100,000〜500,000円以上必ず再発行すべき

0.5カラット以上のダイヤモンドは、鑑定書の有無で数万円〜数十万円の差が出ます。再発行費用は1〜2万円程度なので、十分に元が取れます。

再発行の方法：GIA鑑定書の場合、ダイヤモンドにレーザー刻印されたレポート番号がわかれば、GIAに連絡してレポートのコピーを取得できます。CGLの場合は、ダイヤモンドを直接CGLに送付して再鑑定を依頼します（所要約2〜3週間）。

ブランドジュエリーは高額買取のチャンス

ダイヤモンドや宝石がセットされたブランドジュエリーは、素材価値に加えてブランドプレミアムが上乗せされるため、非常に高額な買取が期待できます。

グラフ素材価格の5〜10倍大粒ダイヤモンドジュエリー

ブランドジュエリーを売却する際は、以下の付属品が揃っているかどうかが重要です。

  • 鑑定書・鑑別書（GIA、CGLなど）
  • 購入証明書・レシート
  • 専用ケース・BOX
  • ギャランティカード（保証書）
  • ショッピングバッグ

付属品がすべて揃っている場合と、何もない場合では、買取価格が20〜50%以上変わることもあります。

ダイヤモンド・宝石買取に関するFAQ

ダイヤモンドの買取価格はどれくらいですか？
ダイヤモンドの買取価格は4C（カラット・カット・クラリティ・カラー）によって大きく異なります。0.3カラットで約3〜8万円、0.5カラットで約8〜20万円、1.0カラットで約30〜100万円が目安です。品質（4Cのグレード）によって同じカラット数でも数倍の差が出ます。
鑑定書がないダイヤモンドも買取できますか？
はい、鑑定書がなくても買取は可能です。ただし、鑑定書がない場合は業者が独自に鑑定するため、鑑定書付きの場合と比べて10〜30%程度買取価格が下がることがあります。GIA（米国宝石学会）やCGL（中央宝石研究所）の鑑定書があると最も高く評価されます。
小さいダイヤモンド（メレダイヤ）も買取できますか？
0.1カラット以下の小さなダイヤモンド（メレダイヤ）は、1石あたりの買取価格は非常に低い（数百円〜数千円程度）ですが、台座の金やプラチナの素材価値で買取されます。メレダイヤが多数ついたジュエリーなら、台座の金属価値と合わせると意外な金額になることもあります。
ブランドジュエリーは高く買い取ってもらえますか？
はい、ハリーウィンストン、カルティエ、ティファニー、ヴァンクリーフ＆アーペルなどのブランドジュエリーは、素材価値に加えてブランドプレミアムが上乗せされます。付属品（ケース、保証書、購入証明書など）が揃っていると、さらに高値がつきやすくなります。金属買取専門店とブランド買取専門店の両方に見積もりを取りましょう。
ダイヤモンド以外の宝石も買取できますか？
はい、ルビー、サファイア、エメラルド、真珠（パール）、アレキサンドライト、タンザナイト、オパールなど多くの宝石が買取対象です。ただし、宝石の種類や品質によって買取価格は大きく異なります。鑑別書があると正確な評価がされやすく、高値がつきやすくなります。

専門スタッフが答える 金買取Q&A

買取の実務でよくいただく質問を、公開情報と当サイトの相場データにもとづいてまとめました。

Q.金の買取価格はどうやって決まりますか？
A.金製品の買取額は基本的に「その日の金相場 × 重量(g) × 純度の比率 × 業者ごとの買取係数」で決まります。たとえばK18なら金の含有率が75%、K14なら58.5%です。相場は毎営業日変動するため、売却日の相場が価格に直結します。本ページ上部に当日の参考相場と推移を掲載しています。
Q.査定や売却に必要なものはありますか？
A.多くの業者で、古物営業法に基づく本人確認書類（運転免許証・マイナンバーカード・パスポート等）の提示が必要です。買取金額が大きい場合や1回の取引で所得が一定額を超える場合は税務上の取り扱いに関わることもあるため、不安な場合は事前に各業者・税務署で確認してください。
Q.複数の業者で査定額は変わりますか？
A.変わります。地金部分の評価は相場連動でどこも近くなりますが、業者ごとの買取係数・手数料・キャンペーン・デザインや付属品の評価で最終額に差が出ます。1社だけで決めず、複数社の相見積り（一括査定を含む）を取ると高値を取りやすくなります。
Q.業者を選ぶときに見るべきポイントは？
A.買取価格（係数）だけでなく、手数料の有無、査定スピード、出張・郵送の対応エリア、キャンセル時の費用、古物商許可や運営会社の明示といった信頼性も確認しましょう。価格と安心感のバランスで選ぶのが失敗しにくい選び方です。
Q.「高価買取」をうたう広告は信用できますか？
A.表示はあくまで上限や条件付きのことが多く、実際の手取りは手数料や状態評価で変わります。広告の数字だけで決めず、複数社の実際の査定額を“手取りベース”で比較するのが安全です。

※ 記載は一般的な目安です。買取の可否・価格・必要書類は各業者の規定により異なります。最終的な条件は各社へご確認ください。

まとめ

ダイヤモンドの買取価格は4C（カラット・カット・クラリティ・カラー）で決まります。同じカラット数でも品質によって数倍の差が出るため、鑑定書の有無が非常に重要です。

ブランドジュエリー（ハリーウィンストン、カルティエ、ティファニーなど）は素材価値の数倍〜10倍以上のプレミアムがつくことがあり、付属品が揃っているかどうかでさらに大きな差が出ます。

ダイヤモンド・宝石を高く売るポイントは以下の3つです。

  1. 鑑定書・付属品を揃える（GIA、CGLの鑑定書は必須級）
  2. 宝石鑑定に強い業者と金属買取業者の両方に見積もりを取る
  3. ブランドジュエリーはブランド買取専門店にも見積もりを取る

まずはヒカカク！の一括査定で複数業者の見積もりを比較してみてください。査定は無料です。
