【2026年最新】金歯の買取相場ガイド — 1本いくら？売り方と注意点

最終更新: 2026年4月15日

「抜いた金歯が引き出しに眠っている」「亡くなった家族の金歯がある」「金歯って売れるの？」——実は、金歯は買取業者で売却できます。しかも、2026年の金価格高騰により、想像以上に高値がつくことがあります。

金歯はジュエリーと同じ金合金で作られており、K14〜K20（純度58.5%〜83.3%）の金が含まれています。見た目は歯の形をしていますが、買取業者は「金としての素材価値」で査定するため、状態や見た目は関係ありません。

金歯は売れるのか？ — 結論：売れる

金歯は間違いなく売れます。大手買取業者のほとんどが金歯の買取に対応しています。

金歯に使われている金は、ジュエリーに使われている金と同じ本物の金合金です。歯科用金合金はK14（純度58.5%）からK20（純度83.3%）のものが一般的で、中にはK18やK20の高純度のものもあります。

買取業者は金歯を溶かして精錬し、純金として再利用します。そのため、歯の形をしていても、セメントやレジンが付着していても、変色していても問題ありません。

よくある誤解：「金歯は不衛生だから買い取ってもらえないのでは？」と心配する方がいますが、全く問題ありません。買取業者は超音波洗浄で付着物を除去し、その後溶解・精錬するため、衛生面の心配は不要です。

金歯の種類別解説 — インレー・クラウン・ブリッジの違い

金歯と一口に言っても、治療の種類によって形状・大きさ・重量が異なります。それぞれの特徴を理解しておくと、買取価格の目安がわかります。

インレー（詰め物）

虫歯を削った後に詰める小さな金属片です。歯の一部を覆うだけなので、重量は0.5〜1.5g程度と最も軽い。奥歯の咬合面（噛む面）に使われることが多く、外見からは見えにくいものです。K14で0.5gの場合、買取価格は約4,500円です。

クラウン（被せ物）

歯全体を覆うキャップ状の金属です。歯の形をしているため、重量は1.5〜3g程度と中程度。前歯に使われることは少なく、奥歯に多く見られます。K14で2gの場合、買取価格は約17,800円です。

ブリッジ（連結冠）

欠損した歯の両隣の歯を支えにして、3本以上の歯を連結した金属です。重量は3〜15g以上と最も重く、買取価格も高額になります。大型のブリッジ（5本以上連結）は10万円以上になることも。K14で8gの場合、買取価格は約71,200円です。

歯科医院での金歯の外し方 — 5つのステップ

「口の中の金歯を売りたいけど、どうやって外すの？」という方のために、歯科医院での手順を解説します。

  1. ステップ1：歯科医院を予約する

    かかりつけの歯科医院に電話し、「金歯の被せ物を外したい」と伝えます。治療の一環として対応してもらえます。

  2. ステップ2：レントゲン撮影・診察

    金歯の下の歯の状態を確認するためにレントゲンを撮影します。歯の状態によって、新しい被せ物を入れるか、抜歯するかの方針が決まります。

  3. ステップ3：金歯を外す

    専用の器具（クラウンリムーバー）を使い、接着剤（セメント）を破壊して金歯を外します。麻酔は基本的に不要ですが、状態によっては局所麻酔を使います。所要時間は約10〜20分です。

  4. ステップ4：「金歯を持ち帰りたい」と伝える

    これが最も重要なステップです。言わないと歯科医院が廃棄してしまうことがあります。「外した金歯を持ち帰りたい」と必ず伝えてください。ほとんどの歯科医院で対応してくれます。

  5. ステップ5：新しい治療を行う

    金歯を外した歯には、新しい被せ物（セラミック、銀歯、プラスチックなど）を入れます。保険適用の銀歯なら数千円程度です。

費用の目安：金歯を外して新しい被せ物を入れる治療費は、保険適用で3,000〜5,000円程度（3割負担）です。金歯の買取価格がこれを大きく上回るため、トータルではプラスになることがほとんどです。

亡くなった家族の金歯を売る場合の手続き

故人の金歯を売却するケースは珍しくありません。特別な法的手続きは基本的に不要ですが、以下の点を確認しておきましょう。

火葬後に金歯が残る場合

金の融点は約1,064度Cで、火葬の温度（800〜1,200度C）によっては溶けずに残ることがあります。火葬後、遺骨とともに金歯が見つかった場合は、遺族が引き取れます。

売却に必要なもの

  • 本人確認書類：売却する方（遺族）の運転免許証やマイナンバーカード
  • 故人との関係の確認：業者によっては戸籍謄本や住民票を求められることがある（稀）
  • 金歯そのもの：そのままの状態でOK。洗浄は軽く水洗い程度で十分

多くの場合、遺族の本人確認書類だけで売却可能です。「故人のものですが買取できますか？」と事前に電話確認しておくとスムーズです。

相続税について

金歯は相続財産に含まれる可能性がありますが、少額であれば実務上は問題になることはほとんどありません。高額（数十万円以上）になる場合は、相続財産に含めて申告することが望ましいです。

金歯の純度 — なぜK14〜K20が使われるのか

歯科用金合金にK14〜K20が多い理由は、口腔内での耐久性と加工性のバランスにあります。

  • K24（純金）：柔らかすぎて噛む力に耐えられない。歯科用には不適。
  • K20〜K18：適度な硬さと耐食性があり、高品質な歯科治療に使用。自費診療で使われることが多い。
  • K14〜K16：硬さと耐久性のバランスが良く、最も一般的な歯科用金合金。
  • 金パラ（金12%）：保険適用の治療で最も多く使われる。安価だが金の含有率が低い。

自分の金歯の純度がわからない場合でも心配ありません。買取業者がX線蛍光分析装置で正確に純度を測定してくれます。測定は無料で、数秒〜数分で結果が出ます。

金歯の売り方 — 具体的な手順

ステップ1：金歯を手元に用意する

金歯が既に外れている場合は、そのままでOKです。まだ口の中にある場合は、以下の方法で外します。

  • 治療のついでに外す：歯科医院で再治療（新しい被せ物に交換するなど）の際に、古い金歯を返してもらいます。「金歯を持ち帰りたい」と伝えれば、ほとんどの歯科医院で対応してくれます。
  • 抜歯の場合：抜歯時に金歯がついたまま抜けた場合も、持ち帰りを希望すれば返してもらえます。

重要：金歯を売る目的だけで健康な歯の金歯を外すのはおすすめしません。あくまで「治療のついで」や「既に外れたもの」を売却しましょう。

ステップ2：軽く洗浄する

金歯を持ち込む前に、水やぬるま湯で軽く洗い、付着した汚れやセメントの大きな塊を取り除いておくと査定がスムーズです。ただし、完璧に洗浄する必要はありません。細かいセメントやレジンは業者が超音波洗浄で除去します。

ステップ3：買取業者に持ち込む

金歯を買取業者に持ち込みます。店頭買取の場合は予約不要のお店が多いですが、事前に電話で「金歯の買取は可能か」確認しておくと安心です。

ステップ4：査定・売却

業者が金歯の重量を測定し、X線分析で純度を判定します。買取価格が提示されるので、納得すれば売却、納得できなければキャンセル（無料）できます。

金歯買取の注意点

1. 複数本まとめて売った方が有利

金歯は1本あたりの重量が小さい（0.5〜3g程度）ため、1本だけでは買取価格が数千円にとどまることがあります。複数本まとめて売ると総重量が増え、業者も高めの単価を提示しやすくなります。

家族や親戚の金歯も一緒に集めて持ち込むのがおすすめです。

2. 金歯以外の歯科用金属との見分け方

種類見た目買取可否買取価格の目安
金歯（K14〜K20）黄金色買取可高い（1gあたり8,900〜12,700円）
金パラ銀色（やや黄色味）買取可中程度（1gあたり約1,800円）
銀歯（銀合金）銀色買取可（安い）低い（1gあたり約100〜200円）
セラミック・レジン白色買取不可—

見た目が金色であれば金歯の可能性が高いですが、正確な判別は買取業者に任せましょう。持ち込めば無料で判定してもらえます。

3. 身分証明書が必要

古物営業法により、買取時には本人確認書類（運転免許証、マイナンバーカード、パスポートなど）の提示が必要です。忘れずに持参しましょう。

4. 故人の金歯の売却

亡くなった方の金歯を売却することも可能です。火葬後に残った金歯を売却するケースも多くあります。特別な手続きは不要で、通常の金買取と同じ流れで売却できます。

金歯買取に関するFAQ

金歯は本当に売れますか？
はい、金歯は買取業者に売れます。金歯に使われている金は本物の金合金であり、K14〜K20（純度58.5%〜83.3%）の金が含まれています。金としての素材価値で買取されるため、見た目や状態に関係なく売却できます。大手の金買取業者のほとんどが金歯の買取に対応しています。
金歯1本の買取価格はいくらですか？
金歯1本あたりの買取価格は約3,000〜15,000円が目安です。金歯の大きさ（重量）と純度によって価格が異なります。クラウン（被せ物）は約1.5〜3gで約13,000〜27,000円、ブリッジ（3本連結）は約3〜8gで約27,000〜71,000円程度です（K14の場合）。2026年4月現在の金価格高騰により、以前より高値で買い取ってもらえます。
金歯を売るにはどうすればいいですか？
まず歯科医院で金歯を外してもらい（治療の一環として）、その後買取業者に持ち込みます。既に外れている金歯はそのまま買取業者に持っていけます。事前に軽く洗浄しておくと良いですが、完璧な洗浄は不要です。業者が超音波洗浄を行った上で、重量測定とX線分析で純度を判定してくれます。
金歯の買取に身分証明書は必要ですか？
はい、古物営業法により、買取時には本人確認書類（運転免許証、マイナンバーカード、パスポートなど）の提示が必要です。これは金歯に限らず、すべての貴金属買取で共通のルールです。
複数の金歯をまとめて売ると有利ですか？
はい、有利です。金歯は1本あたりの重量が小さいため、1本だけでは数千円程度になることが多いですが、複数本まとめて売ると総重量が増え、取引効率が良くなるため業者も高めの単価を提示しやすくなります。家族や親戚の金歯もまとめて持ち込むのがおすすめです。
